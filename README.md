# SBTI

`SBTI` 是一个纯前端静态页面项目，用于学习和演示单页 HTML 网站的结构、样式与交互实现。

## 本地预览

在项目目录运行：

```bash
cd /Users/tianhui/Webstart/SBTI
python3 -m http.server 4173
```

然后打开浏览器访问：

[http://localhost:4173](http://localhost:4173)
